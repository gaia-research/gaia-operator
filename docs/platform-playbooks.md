# Platform Playbooks

This document contains instructions for specific community platforms.

## Reddit Playbook

- **Allowed Interaction Modes**: API read, browser page extraction, local CUA fallback.
- **Rules checked**: Relevance scoring and user intent matching.
- **Engagement constraints**: No automated comments or posts. Draft replies locally and require human approval.
- **Stop triggers**: CAPTCHA, login walls, 429 rate limit statuses, or account suspension/warning prompts.

## Hacker News Playbook

- **Rules**: Research-only mode.
- **Goal**: Observe developer pain points, draft offline responses, and avoid promotional links.

## GitHub Discussions Playbook

- **Goal**: Safe integration tracking. Find issues and discussions. No PR or issue creation in MVP without explicit approval.
